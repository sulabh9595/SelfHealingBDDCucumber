- Maven profiles for different environments and browsers

- Enhanced logging with Logback
- Documentation improvements (README, CONTRIBUTING)
- Allure report artifacts in CI/CD

### Changed
- Updated pom.xml with additional dependencies and plugins
- Enhanced project structure documentation

### Fixed
- Fixed logging configuration

- BDD framework with Cucumber 7.34.3

- Page Object Model implementation
- Allure reporting
- YAML test data management
